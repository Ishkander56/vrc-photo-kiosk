# VRC Photo Kiosk
A web-based "multi layered" screenshot compositor for VRChat.

## Features

* Load separate image files created using VRChat's Multi Layered photo function.
* Reposition, resize, and rotate layers.
* Modify brightness, saturation, contrast, opacity, and color blending.
* Blur individual layers.
* Add text.
* Swap between presets to quickly try out different moods for your photo.
